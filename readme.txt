Streamline your media library with folders: this plugin creates media library categories.


== Description ==

TinyFolders allows you to organize your Media Library into folders or categories. Keeping your Media Library organized is essential, and this plugin will assist you in categorizing your images. To create a category, navigate to Media > Categories and set up a new category. Then, head to Media > Library to move your images into the designated category.

When you add or edit an image in the Media Library, you can choose the category or categories for that image.


== Installation ==

* go to the Wordpress admin console
* go to "Plugins" -> "Add New"
* search for "TinyFolders"
* Click "Install Now"
* Click "Activate"
